# Foody
## Description
###An Alexa skill to find the nutrition information for a food.
An Alexa skill that calculates the nutrition information for a specific food. The skill also displays additional workout details for the food using skill cards in the companion Alexa app.

## Usage
* Open the Alexa app on your phone.
* Search for Foody.
* Enable this skill from your Alexa app.

## Sample Utterances
* Alexa, ask Foody for nutrition on bread
* Get workout for pasta
* Milk

## Source
All nutrition information is sourced from the USDA nutrition database.
